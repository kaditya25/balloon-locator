# Lab 5

## Running the code
First, build the code.
```bash
mkdir build
cd build
cmake ..
make
```

The executable is but in the exe directory. It requires one input: the path to an
image to display.
```bash
cd exe
./lab5 ../images/frame00077.jpg
```
